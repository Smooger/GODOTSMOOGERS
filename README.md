# GODOTSMOOGERS

Broderick's Bathroom Dash is a 2D platformer prototype built with Godot 4.

## Current playable features

- Seven themed stages that auto-progress when you reach each goal.
- Theme cycle: Desert Ruins, Ice World, Moon Base, Underwater Base, Rainbow Fortress, Dinosaur Island, and a final T-Rex showdown.
- Enemy combat with both stomp attacks and laser blasts from powerups.
- Laser powerup grants limited shots shown in HUD.
- Final level includes a boss-style T-Rex enemy with higher health.

## Controls

- `A / D`: move
- `Space`: jump
- `F`: shoot laser (when powered up)
- `R`: retry after losing or winning
